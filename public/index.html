使用说明：

  1. 选择要转换的文件类型（Word/Excel/PowerPoint）
  2. 点击"选择文件"按钮上传文档
  3. 点击"转换"按钮开始转换
  4. 转换完成后自动下载PDF文件
  5. 如果转换失败，请检查服务器日志获取详细信息

系统要求：

需要安装 LibreOffice 才能正常使用转换功能

Windows: 下载安装

macOS: brew install libreoffice
